# Att starta projektet

- Klona projektet

* git clone <repo-url>

- Installera beroenden

* npm install

- Cd till projektet vi konsolen

* cd "house_r3_project"

- Starta projektet

* npm run dev

# Techstack

React (Vite eller Next – välj rätt)

- Vanlig CSS

- Flexbox + Grid

- Responsiv design via media queries (Mobil)

- ESLint

# Motivering av val

React användes eftersom jag är bekväm med det och det möjliggör en tydlig komponentstruktur även i små projekt.

Vanlig CSS istället för t.ex. Tailwind då jag ville visa ren förståelse för CSS-grunder, layout och responsivitet utan frameworks.

Flexbox & Grid gav mig kontroll över layouten och gjorde det enkelt att efterlikna designskissen.

Media queries användes för mobilanpassningen då det passar bra när man inte använder ett större CSS-ramverk.

# Responsiv design

- Grid-layouten 1 kolumn på mobil

- minska padding/marginaler där det behövdes

- Anpassade bilder och textstorlekar.

-Fokus var att inte överkomplicera det utan hålla sig till CSS-grunder.
